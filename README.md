🛒 Midterm Output – Food Ordering Website
📖 Short Description

This project is a static food ordering website developed as a midterm academic requirement. It features multiple web pages including a homepage and order page, styled with CSS and enhanced with graphical assets.

The system simulates an online food ordering interface where users can browse menu items, view product images, and navigate between pages. The design focuses on layout structure, UI styling, and proper file organization.

✨ Features / Functionality

🏠 Homepage interface

📋 Order page layout

🖼️ Product image display

🛒 Cart icon and add-to-cart UI elements

👤 Account interface icon

📱 Social media icons integration (Facebook, Instagram)

🎨 Styled using external CSS

📂 Organized file structure with assets

🛠️ Technologies Used

HTML – Web page structure

CSS – Styling and layout design

PNG Images – UI elements and product graphics

🚀 Installation / Usage Instructions
Requirements

Any modern web browser

Steps to Run

Clone or download the repository.

Open the project folder.

Double-click Homepage.html
OR
Open it in your preferred web browser.

No additional setup or installation is required.

🎯 Purpose

This project was created as a midterm output to demonstrate understanding of:

Web page structuring using HTML

Layout styling using CSS

File and asset organization

Basic user interface design principles

It serves as a foundational project in web development.
